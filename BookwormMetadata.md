# Metadata in Bookworm #

This is a space for public comment on Bookworm's support of metadata.

## Current support ##

### Dublin Core/OPF metadata ###

  * dc:identifier
  * dc:publisher
  * dc:language
  * dc:subjects
  * dc:rights
  * dc:creator
  * dc:description
  * dc:date

### Bookworm-specific ###

  * Date added to library
